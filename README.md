# cancer-patient-analysis-survival-dashboard
## Overview
This project is a *Cancer Patients Analysis & Survival Dashboard* created using *Microsoft Power BI Desktop*.  
It provides insights into *patient demographics, cancer types, treatments, and survival rates*, helping healthcare professionals monitor outcomes and trends.

## Features
- *Patient Demographics:* Age, gender, location distribution.
- *Cancer Type Analysis:* Frequency, stages, and affected organs.
- *Survival Analysis:* Kaplan-Meier survival curves, treatment outcomes.
- *Interactive Visuals:* Filters, charts, and slicers for detailed exploration.
- *Treatment Monitoring:* Effectiveness of therapies and follow-up statistics.

## Dataset
- Source: Sample cancer patient data (CSV/Excel)
- Key Fields:
  - Patient ID, Age, Gender
  - Cancer Type, …
